# Setup Web Application - Prerequisite

Before deploying the web application, ensure:

- Admin/API backend is fully installed and reachable
- Correct API base URL is available
- Domain and SSL are ready for web frontend
- Build environment (Node.js + package manager) is installed

## Buyer Checklist

- [ ] Read web setup guide from CodeCanyon ZIP
- [ ] Confirm API health endpoints are reachable
- [ ] Confirm CORS and HTTPS policies are ready
